# New York Code School Admin Panel
Work in progress: Admin panel for a fictitious coding school whereby the administrator can create users, cohorts and courses as well as edit and destroy as needed.

# Technologies Used
- Ruby 2.3.0
- Rails 5.2.0
- PostgreSQL 10.5
- SASS 3.5
- Bootstrap 4
- JQuery
